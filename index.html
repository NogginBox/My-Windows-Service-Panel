My Windows Services Panel

Lets you quickly start and stop the Windows Services you care about

Download .msi installer View on GitHub Download .zip source code

Choose the services you stop and start frequently

Fed up with having to browse through the full list of Windows Services to find the few the one that you're trying to start.

Manage Windows services you frequently stop and start like MySQL, MS SQL Server, IIS and MSMQ.

Requirements

This program requires a Windows computer with .NET 4.6 installed. You can get the latest .NET version here.
